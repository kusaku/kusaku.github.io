MP3 Tagger (2002)

A tiny Turbo Pascal toolkit for reading, editing, and normalizing MP3 ID3v1 tags.

  • File I/O Utilities → - File input/output helpers for MP3 tag data.
  • MP3 Tag Editor → - Command-line ID3v1 tag editor with text normalization.
  • String Utilities → - Text processing helpers used by the tag editor.
