Other Programs (1996-1999)

Mathematical calculators, small games, utilities, and experimental code grouped outside the larger Turbo Pascal sections.

  • Broken Sort Algorithm → - An incomplete sorting algorithm with educationally useful mistakes.
  • Error Function Calculator → - Numerical approximation for the error function.
  • Performance Test with Error Function → - A debugging and performance variant of the error function calculator.
  • Polynomial Root Solver → - A small polynomial equation solver.
  • Reaction Test Game → - A graphical reaction-time game.
  • Reaction Test Game Variant → - A slightly different version of the reaction test.
  • Terminal Login Logger → - A terminal session logger using DOS date/time and file operations.
