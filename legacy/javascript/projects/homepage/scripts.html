Java скрипты от kusaku

В разном времени и месте я увлекаюсь написанием java - скриптов. Некоторые из них ты можешь найти на www.physfac.nm.ru, остальные выложены здесь (по возможности с комментариями). ВСЕ СКРИПТЫ ТОЛЬКО ПОД IE! Если ты знаешь javascript, то без особого труда сможешь переделать их и для NN. А NN я не уважаю. Если тебе нужен скрипт, то пиши мне и я с удовольствием тебе его напишу.

Мои скрипты:

Tetris Пародия на тетрис. Основная особенность - компьютер может играть сам (довольно неплохо, по сравнению со мной :). Top score хранятся в пирожках. [part of www.physfac.nm.ru site]
JS debug Отладчик, который может быть очень полезен тем, кто при написании скриптов активно пользуется об'ектами. Только под IE. Позволяет просматривать иерархию об'ектов и значений и следить за изменением значений. В большом поле - имена-исключениия при построении иерархии, в первом поле вводишь урль, нажимаешь prepare, пишешь имя нужного объекта (регистр как всегда важен), жмешь explore. Для просмотра простых типов жмешь trace и время обновления рядом (в миллисекундах). Настройки сохраняются в пирожках.
Говорящий хрен Читает текст вслух, запоминает позицию, подсвечивает читаемое место и др.
Опросник Движок для вопросника, может задавать различные типы вопросов (без ветвления), настройки и ответы сохраняются в пирожках (требует доработки).
Шизотест Тест (кажется на профпригодность), на движке опросника. Более 500 вопросов! обработкой результатов заведует DOS-программа,так что алгоритм обработки мне не известен, но можно присылать мне анкеты :) (требует доработки)
Декодер Раскодирует шифрованные java - скрипты. Шифровать умеет, но только тело шифровки (при дешифровки надо вставлять только ТЕЛО! Не знаешь как выглядит заголовок - удаляй по одному символу в начале (заголовок короткий!))
Шифратор Позволяет шифровать с ключем и дешифровать любой текст
HL menu Менюшка, которая использует фильтры IE для эффектов нажатия, в стиле Half-Life (требует доработки)
text menu Менюшка, которая использует фильтры IE для эффектов нажатия (только текст) (требует доработки)
menu Меню, которое используется на этом сайте. Скачай заодно какую-нибудь страницу сайта, чтобы было яснее.
arranger Оптимизатор расположения файлов при записи болванок. Метод заключается в слепом переборе, так что если скрипт долго не отвечает, то вылезет окно с предложением снять его. Если снять, то придется нажать "display", чтобы увидеть промежуточные результаты. Не стоит пытаться оптимизировать больше 20 файлов для более 3-х болванок (да это можно и руками сделать :). Ограничение - 8 дисков, но если файлы разбить на группы, то оно снимается. "fff" - для теста.
teorver Скрипт для листания картинок (картинки в комплект не входят) [part of www.physfac.nm.ru site]
regionmaker Создатель областей картинок для тега AREA. Все просто и ясно. [part of www.physfac.nm.ru site]
digits Скрипт, который рисует таблицу Пифагора для систем исчисления до 36 включительно (простой конвертатор)
chasBall Волейбол 1x1 и возможность сразиться с компом! [part of www.physfac.nm.ru site]
brown Броуновское движение частицы [part of www.physfac.nm.ru site]
maxwell распределение Максвелла-Больцмана [part of www.physfac.nm.ru site]
