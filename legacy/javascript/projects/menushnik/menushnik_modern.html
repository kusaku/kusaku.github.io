Menushnik Modern

A modernized hierarchical navigation menu system with unlimited nesting levels. Hover over menu items to reveal submenus.

Features

  • Unlimited nesting levels
  • SVG icons for all menu items
  • Automatic light/dark mode support
  • Pure CSS hover effects (no JavaScript required)
  • Accessible markup with semantic HTML
  • Smooth transitions and animations

Technical Details

This menu system uses modern CSS features including:

  • CSS custom properties (variables) for theming
  • :has() pseudo-class for parent selectors
  • CSS transitions for smooth animations
  • SVG icons with currentColor for automatic color inheritance
  • Media queries for dark mode detection
